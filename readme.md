# 文档

### 提交
+ `git add`
+ `git commit -m ""`
+ 当前状态 `git status`
+ 查看修改内容 `git diff <file>`
+ `git push`

### 版本回退
+ `git log --pretty=oneline`

### 分支命令
+ 查看分支  `git branch -a`
+ 创建分支  `git branch <name>`
+ 切换分支  `git checkout <name>`
+ 创建+切换分支 `git checkout -b <name>`
+ 删除分支 `git branch -d <name>`
+ 发布分支到远程 `git push origin <name>`
+ 删除远程分支 `git push origin --delete <name>`
